# XSMB 双尾遗漏策略分析报告

## 策略概述

基于越南北方彩票（XSMB）特等奖最后两位数字的统计分析，发现尾数遗漏规律可产生正收益。

### 核心策略

**双尾遗漏策略**：同时投注遗漏天数排名第1和第2的尾数。

| 参数 | 值 |
|------|-----|
| 训练窗口 | 365天 |
| 第1名条件 | 遗漏≥15天 且 与第2名差距≥5天 |
| 第2名条件 | 遗漏≥15天 且 与第3名差距≥5天 |
| 投注方式 | 满足条件的尾数各投10注（10-20注/天） |

---

## 策略效果（历史回测）

### 核心指标

| 指标 | 单尾策略 | 双尾策略 | 提升 |
|------|----------|----------|------|
| 年利润 | 3,105,943 VND | 3,663,774 VND | **+18%** |
| 最长连续不中 | 36天 | 22天 | **-40%** |
| 命中频率 | 7.2天/次 | 6.1天/次 | **+18%** |
| ROI | +16.0% | +11.7% | -27% |

### 财务模型

- 每注成本：10,000 VND
- 中奖赔付：840,000 VND（8.4倍）
- 盈亏平衡命中率：11.9%
- 实际命中率：~16.5%

---

## 已验证的优化方向

### ✅ 有效
| 优化 | 效果 |
|------|------|
| 双尾策略 | 利润+18%，风险-40% |
| 遗漏≥15天 | 最优阈值 |
| 差距≥5天 | 最优阈值 |
| 365天窗口 | 比短窗口更稳定 |

### ❌ 无效
| 优化尝试 | 原因 |
|----------|------|
| 头数过滤 | 分布均匀，无规律 |
| 奇偶/大小过滤 | 50/50分布 |
| 第2名单独调参 | 可能过拟合 |
| 等待连续不中 | ROI高但利润降低 |
| 星期/月份效应 | 样本量不足 |

## 使用说明

1. 每天查看 `index.html` 获取投注信号
2. 绿色 = 第1名尾数信号
3. 蓝色 = 第2名尾数信号
4. 满足条件的尾数各投10注
